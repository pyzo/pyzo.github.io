Pyzo - Python to the people¶

Pyzo is an interactive computing framework based on Python, an easy and powerful programming language. It’s mission is to provide a computing environment aimed at doing science and building professional applications, that is easy to obtain, easy to use, and free.

  • The open source Python community has great potential to deliver high quality software for use in science and engineering.
  • However, the software landscape is fragmented, packages are not always sufficiently documented, and there is little professional support.
  • Pyzo aims to solve this by wrapping the right packages into a computing framework that feels coherent. We distribute the free Pyzo distro, offer paid support, and improve coherence by helping improve documentation and standards of the different packages.
